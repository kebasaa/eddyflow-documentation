# Videos

The following video tutorials will help familiarize you with EddyFlow.

![thumbnail](https://embed-ssl.wistia.com/deliveries/ef21c6971452c31859c8cbcd72c977e2d7e39665.jpg) Learning EddyFlow This one-minute video introduces EddyFlow and provides tips for learning EddyFlow.

![thumbnail](https://embed-ssl.wistia.com/deliveries/76e237df89d4ae3ca81462a2a34d54937055c795.jpg) Processing .ghg files in EddyFlow Express Mode This four-minute video shows how to process .ghg files in Express Mode. .ghg files are logged by the LI-7500A/RS/DS and LI-7200/RS gas analyzers.

![thumbnail](https://embed-ssl.wistia.com/deliveries/94207783817c56471de31de2249ed5a62cc75ad2.jpg) Abate EddyFlow Processing Time: Using Results from a Previous Run This four-minute video introduces six ways to reduce processing time by using results from a previous run.

![thumbnail](https://embed-ssl.wistia.com/deliveries/66a176e5628587a6b144690546913c2834bf6ddb.jpg) Advanced Settings: EddyFlow Processing Options This four-minute video provides an overview or Processing Options under Advanced Settings.

![thumbnail](https://embed-ssl.wistia.com/deliveries/a554048839e134dca496f71b4ce64328a5f38124.jpg) Advanced Settings: EddyFlow Spectral Corrections In this eight-minute video, we introduce the Spectral Corrections under Advanced Settings and provide some tips for getting satisfactory results.

![thumbnail](https://embed-ssl.wistia.com/deliveries/8374d22776f558f190e375ceaddf764d49534c71.jpg) Advanced Settings: Statistical Analysis in EddyFlow This one-minute video is an overview of Statistical Analysis options under Advanced Settings.

![thumbnail](https://embed-ssl.wistia.com/deliveries/17ae99a27324cb5c0492d8c5b79f4c13075ea1ee.jpg) Advanced Settings: Output Files from EddyFlow This one-minute overview covers options for output files provided with each project.
