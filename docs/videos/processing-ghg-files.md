# Processing .ghg Files

This four-minute video shows how to process .ghg files in Express Mode. .ghg files are logged by the LI-7500RS and LI-7200RS gas analyzers and provide the simplest way to compute fluxes in EddyFlow. Download a [sample data set](https://app.boxenterprise.net/s/4omma8p9bd3pfhctbqjxhjh04960x06k) and follow along.
